    # smart-cli

    smart-cli，为了方便开发小程序时可以快速生成小程序开发目录，CSS语法兼容等目的，而存在的一个脚手架命令行工具。

    注意

    smart-cli 安装要求

    • Node.js (opens new window) >= 9
    • npm源需要是百度源，http://registry.npm.baidu-int.com/

    # 安装

     npm install @baidu/smart-cli -g --registry=http://registry.npm.baidu-int.com

    # 初始化

     smart init <app_name>

    初始化完成后，会快速下载一个模板目录到本地。

    # 开始

     cd <app_name>
 npm install
 npm run dev

    命令执行完成后，在本地产生一个符合原生小程序规则的 dist (opens new window) 目录，可使用百度开发者工具 (opens new window)打开。

    # 特点：

    • 基于百度原生小程序 (opens new window)；
    • 为使用原生语法开发百度小程序提供一套快捷便利的的命令行工具；
    • 可以快速生成符合百度原生小程序开发的目录结构；
    • 支持本地和远程MOCK，方便联调；
    • 支持stylus，less等CSS预处理；
    • 支持px2rpx单位转换；
    • 配置化，方便扩展
