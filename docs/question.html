# 常见问题

# CLI安装不成功

  1. npm安装源registry设置错误，引起安装不成功

注意

目前npm安装源主要是针对百度内网源

处理方案可参考：

  • 通过config命令
npm config set registry <baidu_registry>
  • 修改.npmrc文件
# .npmrc
registry=<baidu_registry>
  • 安装时使用 --registry
$ sudo npm install -g @baidu/smart-cli --registry=<baidu_registry>
  • 使用nrm (opens new window)管理工具管理registry地址
npm install -g nrm
nrm add baidu <baidu_registry>
nrm use baidu
  1. 由fecs设置引起的安装不成功

具体问题提示可能如下：

处理方案可参考：

npm config set unsafe-perm=true
sudo npm install fecs -g
sudo npm install -g @baidu/smart-cli --registry=<baidu_registry>
